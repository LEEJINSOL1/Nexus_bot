How to Use

Step-by-step guide to set up your copy trading bot

Go to API Management

Log in to Bybit and navigate to API Management. Click "Create New Key" or "API" in the Tools menu.

Select API Key Type

Choose "System-generated API Keys" for HMAC encryption. This is the recommended option.

Configure Permissions

Set permissions to "Read-Write" and enable "Unified Trading" with Contract, USDC Contracts, and SPOT trading permissions. No IP restriction needed.

Save Your API Keys

Copy your API Key and API Secret immediately. The secret key will only be shown once. Keep them secure.

Find Your UID

Your UID can be found in your Bybit account profile. It's displayed next to your profile icon.

Start the Bot

Open the Telegram bot and click "Exchange API" button, then select "Bybit".

Choose Trading Mode

Select "Real Trading" for live trading or "Demo Trading" to practice with virtual funds.

Enter API Credentials

Enter your Bybit API Key and Secret Key. The bot will verify your credentials automatically.

Enter Your UID

Enter your Bybit UID to complete the setup. For demo trading, UID verification is not required.
