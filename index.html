How to Use

Step-by-step guide to set up your copy trading bot

1. After logging in to Bybit, select Tools > Demo Trading

Demo copy trading is available for free to everyone. Real copy trading is only available to users who signed up through Nexus's Bybit referral link.

2. Navigate to API Management

Click on your profile icon in the top right corner, then select "API" from the dropdown menu to access the API Management page.

3. Create New API Key

On the API Management page, click the "Create New Key" button to start the API key creation process.

4. Select System-generated API Keys

Choose "System-generated API Keys" option. This uses HMAC encryption and is the recommended method. You will receive both a public and private key pair.

5. Configure and Generate API Key

Enter a name for your API key, set permissions to "Read-Write", enable "Unified Trading" with Contract, USDC Contracts, and SPOT trading permissions, then click "Submit" to generate your keys.

6. Save Your API Credentials

After generation, copy both your API Key and API Secret immediately. Save them securely in a text file or password manager. The secret key will only be shown once and cannot be retrieved later.

1. Start Nexus Bot

Open the Telegram bot and click the "Exchange API" button to configure your API keys. This is the first step to set up your copy trading account.

2. Select Bybit Exchange

Currently, Bybit is the only officially supported exchange. More exchanges will be supported in the future, expanding your trading options.

3. Choose Trading Mode

Select "Real Trading" for live trading or "Demo Trading" to practice with virtual funds. Then enter your Bybit API Key, Secret Key, and UID to complete the authentication.

4. Set Up Copy Trading

Click "Copy Trading" and enter the wallet address of the trader you want to copy trade from. You can find wallet addresses on platforms like Hyperliquid or Binance Leaderboard.

5. Access Copy Management

Click "Manage Copy" to access your copy trading settings. Use "View Positions" to check the current positions of the trader you're copying from.

6. Set Your Copy Ratio

Click "Set Ratio" to configure your copy trading ratio. For example, a ratio of 0.001 means if the target opens a 1 BTC position, you will open a 0.001 BTC position. Adjust this based on your risk tolerance and available capital.

7. Configuration Complete

Your copy trading configuration is now complete. Return to "Manage Copy" to review your settings and prepare to start copying trades automatically.

8. Start Copy Trading

Click the "Start Copy" button to begin automated copy trading. The bot will start synchronizing positions within approximately 30 seconds, and your trades will begin automatically.

9. Monitor Your Positions

Your copy trading bot is now running 24/7, automatically mirroring trades even while you sleep. Log in to your Bybit account to verify your positions and monitor your trading performance in real-time.
